# ClimateVision

**Определение сезона и месяца по фотографии, температуре, геолокации и высоте над уровнем моря**
---

## О проекте

ClimateVision — это ML-проект, который по входным данным определяет:

- **Сезон** (весна, лето, осень, зима)
- **Месяц** (усложнённая версия)

**Входные данные:**
- Фотография (любой пейзаж/улица)
- Температура воздуха (°C)
- Город **или** координаты (широта, долгота)
- Высота над уровнем моря (м)

**Выходные данные:**
- Сезон + месяц (например: *"Весна, апрель"*)
